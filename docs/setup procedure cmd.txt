
Decide on which PC to connect the EEG device a to activate the LSL stream. Can be different PC than the VBS PC.

Check the :
.\config\hardware\freeeeg.yaml

Activate the LSL stream on the selected PC.

PS D:\F2H_code\GH\EEGsuite> python -m src.main server -c .\config\hardware\freeeeg.yaml


Run the VBS sweep
PS D:\F2H_code\GH\EEGsuite> python -m src.main sweep -d .\config\hardware\freeeeg.yaml -p .\config\protocols\sweep_default.yaml



Analyze the EEG data
PS D:\F2H_code\GH\EEGsuite> python -m src.main analyze -f "G:\My Drive\SharedData\data\raw\260223-1749_FREEEEG32_BOARD_c1_f32_v100.csv" -s 0 -d 60
